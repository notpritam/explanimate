# Session 0001 — 2026-06-10 — Founding session

## The brief

Revamp `excalidraw-diagram-skill` (v1: agent hand-crafts Excalidraw element JSON) into a v2 that is
**purely React code**: the agent designs scenes itself in SVG + HTML + Tailwind, gets animation
libraries (Motion), gets Remotion for browser video playback and MP4 rendering, follows the
card-against-humanity engineering discipline (ABOUTME headers, hooks, build log, conventional
step-by-step commits), gets validated by multiple agents (reviewer, QA, architecture), and installs
as a standard skill. Unique name required.

## What happened, in order

1. **Recon.** Read v1's SKILL.md (the methodology worth keeping: argue-don't-display, isomorphism
   test, evidence artifacts, multi-zoom, render-validate loop) and CAH's CLAUDE.md/hooks/scripts
   (the discipline to mirror).
2. **Named it `explanimate`** — explain + animate. Runner-ups: vistruct, scenesmith, kinexplain.
3. **Architecture** (docs/architecture/overview.md): scenes are code; two animation drivers over
   animation-agnostic primitives (Motion for scenes, frame clock for video); template-copy
   distribution with a per-project studio; mandatory visual verify loop; CAH discipline on the repo.
4. **Repo infra** — husky pre-commit (headers + prettier via lint-staged), commit-msg (commitlint,
   scoped conventional), pre-push (validate), check-headers.mjs. The pre-commit hook reformatted
   its own config on the first commit — proof of life.
5. **SKILL.md + references** — slim entrypoint, four deep-dive docs (design language, diagram
   patterns, motion recipes, remotion video).
6. **The template** — Vite + React 19 + TS + Tailwind v4 + Motion 12 + Remotion 4 studio:
   primitives (Stage/Node/Edge/Label/Dot/Reveal), hash-routed gallery + scene viewer + Player page,
   shoot.mjs (Playwright), reference scene + its video composition twin.
7. **Verification, the honest kind.** Install (remotion lockstep at 4.0.474), typecheck clean,
   build clean. First `pnpm shoot`: **the verify loop caught two real defects in its own reference
   scene** — a card clipped at the stage edge, a label wrapping against the containing block. Fixed
   (one fix landed in the Label primitive itself), reshot clean. Remotion stills at frames
   45/140/290 showed correct deterministic mid-draw states; full 300-frame 1080p MP4 rendered.
8. **Consumer journey** — `init.mjs` scaffolded into /tmp, installed, typechecked, shot green;
   clobber guard verified.
9. **Agent panel** — first architect agent died on a session limit (logged); post-build panel
   relaunched: a skill-consumer reviewer (cross-checks every doc claim against template source) and
   a QA reviewer (bug hunt with a live studio). Findings → fixes → commits.

## The design insight worth remembering

The composition's first draft duplicated the scene layout with per-element font-size overrides —
unmaintainable. The fix that made video and scene true twins: `Stage` takes an explicit `scale`
prop (compositions pass `scale={1.5}` to map 1280×720 onto 1920×1080), which is also the
frame-determinism fix (ResizeObserver auto-fit is async and can't be trusted under a frame clock).
One prop solved aesthetics and correctness at once.
